User Login Functionality
Description: Verify that users can successfully log in to the application using valid credentials and are redirected to the dashboard.

Prerequisites:
- Valid user account exists in the system
- User has correct email and password
- System is accessible via web browser

Acceptance Criteria:
- User can access the login page
- System validates email format
- System validates password requirements
- User receives appropriate error messages for invalid credentials
- User is redirected to dashboard after successful login
- Session is created after successful login
